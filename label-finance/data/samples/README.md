# CSV di esempio

Metti qui i file CSV di esempio su cui costruire e testare il motore di import:

- `bandcamp_*.csv` — export vendite da Bandcamp (Tools → Sales report)
- `distributore_*.csv` — report del tuo distributore (DistroKid/Believe/…)
- altri (PayPal, Stripe, banca)

Vanno bene anche file con **pochi dati finti/anonimizzati**: servono solo a
riconoscere il formato delle colonne.
